# The SLP Lab at UNR

**Speech and Language Processing Lab**  
University of Nevada, Reno

## About

The SLP Lab at UNR investigates human speech, linguistic structure, and multilingual language processing using computational and experimental methods.

## Website

This repository contains the source code for our lab website: [https://slp-unr.github.io/lab](https://slp-unr.github.io/lab)

## Research Areas

- Human speech analysis
- Linguistic structure
- Multilingual language processing
- Computational linguistics
- Experimental methods in language research


## Contributing

Lab members can contribute by creating a branch and submitting a pull request. For questions, contact the lab director.
